K-means聚类

Veröffentlicht am 2017-09-10

数据聚类：

K-means算法（K均值）

K均值算法的主要步骤：

1.随机设置K个特征空间内的点作为初始的聚类中心。
2.对于每个数据样例的特征向量，从K个聚类中心里找到距离最近的一个，并把该数据标记为从属这个聚类中心。
3.根据这些数据新分配的类簇，重新对K个聚类中心位置进行计算。
4.如果K个聚类中心位置发生变化，则回到2继续循环，否则迭代停止。

How to use Vim

Veröffentlicht am 2017-03-11

How to use Vim effictivly ?

进入文件

vim filename

若文件不存在会自动创建

进入插入模式的四种形式：

  • i
  • a 在当前光标后追加数据
  • s
  • A 在当前光标所在行行尾追加数据

小范围移动光标：

h:左移
j：下移
k：上移
l：右移

大范围移动光标：

Ctrf F : 向下移动一屏数据
Ctrf B : 向上移动一屏数据
n    G : 移动到第n行
gg     : 移动到第一行
G      : 移动到最后一行

文件退出与保存：

q： 用于只查看数据，未修改文件
q！: 取消所有修改退出
wq : 保存修改并退出
w filename ：将该版本保存到另一个文件下

文件编辑：

x：删除光标所在位置字符
dd：删除光标所在行
dw：删除光标所在位置单词
d$:删除从光标所在位置至行尾
J：删除光标所在行行尾的换行符(拼接行)
