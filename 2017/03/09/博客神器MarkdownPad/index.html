MarkDown语法

Posté le 2017-03-09

  • 总结

    1.为什么要学习markdown

  • 许多网站像github，stackoverflow，简书等等都支持了markdown作为书写规范，而且越来越多的网站正在加入这个行列来，如果你还不会就未免显得太low了。
  • 许多你所看到的博客都是用markdown语法来写的，不仅仅是程序员，markdown也为许多文字爱好者所接受，如果你想打算写自己的博客那就不要犹豫了。

    2.关于markdown你所要知道的

    markdown的由来不是我们所要了解的，如果你真的想知道，那就点击这里吧，真正你所要知道的是：

  1. 用markdown编写的文件后缀名为md(或许你有点熟悉)，md文件是纯文本文件，你可以用几乎任何文本编辑器来编辑它，当然用户体验各不相同。
  2. markdown完全兼容html语法，也就是说如果你直接敲<h1>balabala</h1>那么你一定看不到俩边的<h1>和</h1>(不要问这里我是怎么做到的QAQ)，当然我还没有见到在markdown里面写html的牛人。
  3. markdown编写的文本可以转换为相应的html，pdf，word文本，当然需要借助一些工具，听起来似乎以后可以不用写html了，当然现在我不会在这里教你怎么做，因为这个功能你几乎用不到，最重要的是我也不会(2333333),说不定以后哪一天我会用到，到时再写一篇吧！

    3.在哪里编辑md文本呢？

  • 在线编辑，有很多可以在线编辑md文本的网站，这里贴一个国人开发的在线编辑器MaHua,现在就可以点进去看看。
  • 本地编辑，前面说过几乎所有的文本编辑器都可以用来编写md文本，但是所有人都希望可以实时的看到显示效果，Windows下推荐MarkdownPad,也是我写这篇文章的时候用的，Mac系统下推荐Mou。

  • 4.markdown语法

    标题

    markdown文本中有1到6共六级标题，各个等级标题字体大小不同，对应于文字左边相应数目的#号。
    示例：
    ####### 六级标题
    显示：
    六级标题

列表

有序列表

示例：

1. 第一行列表项
2. 第二行列表项
3. 第三行列表项

显示：

  1. 第一行列表项
  2. 第二行列表项
  3. 第三行列表项

无序列表

示例：

* python
* php
* java

显示：

  • python
  • php
  • java

注意：中间有空格！！！

代码块

强烈推荐使用换行和Tab缩进来显示代码块。

示例:

def add(x, y):
    print(x + y)

分割线

示例: 

***
___
---

显示：

___

超链接

语法：

[link text](url)

没错就是这么简单！这是内联式写法，还有一种参考式写法

[link text][link]

[link]: url

示例：

[baidu](http://www.baidu.com)

显示：
baidu

图片

在前面说过markdown编辑的是纯文本文件，所有你在其中看到的图片都是外部导入的。
语法：

![text](img_url)

示例：

![郭碧婷](https://imgsa.baidu.com/baike/c0%3Dbaike116%2C5%2C5%2C116%2C38/sign=70714f858e44ebf8797c6c6db890bc4f/b3119313b07eca807e294f61982397dda1448303.jpg)

显示：

斜体和强调

示例：

*这是斜体文本*  
**这是强调文本**

显示：
这是斜体文本
这是强调文本

网页内的“锚”

当你浏览的页面很长时候，你往往需要一种在当前页面内的导航链接帮你导航到页面内特定位置，这时候你就需要一个anchor，中文翻译为“锚”。
怎样完成这个任务呢？首先在你想要导航至的位置前加上一行<a name=”anchor”></a> ，让后在网页开头部分加上一个文本链接即可，其url改为#anchor即可。
示例： 

[reach the summary](#summary)
<a name="summary"></a>

显示：
reach the summary

差点忘了还有转义

当你想写一个#加一行文本时发现怎么也显示不了时，这时候你可能需要用到转义字符了\了，只要你对计算机有一点了解就应该知道这个\的用法，把它加在需要转义的字符前面就可以了，你可能又会发现你还是无法显示出<h1>这样的html元素标签，如果你真的是用来显示html标签的话，那么你一定听叫做说过字符实体，忘记了？没关系，点击这里回顾一下。
示例：

\#一行你并不想作为标题的文本
&lt;a&gt;这并不是一个链接&lt;\a&gt;

显示：
#一行你并不想作为标题的文本
<a>这并不是一个链接<\a>

5.总结

结束了，markdown常用语法就这么多，还有一丢丢我没有介绍，因为我保证你用用不到。最后布置一个简单的小任务，写一个像下面这样的图片链接，直接回复在评论区即可，怎么样很简单吧！
