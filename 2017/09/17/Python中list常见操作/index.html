2.list扩展之append()和extend()

a.append(b)作用为将b添加到列表a的末尾作为a的一个元素，不管b的结构是什么样的。

a.extend(c)中c必须为一个可迭代iterable对象，将c中的元素一个个抽出来分别加入到a中，其中c可为list，tuple，set，str等。

4.花式切片

语法：items[start:end:step]
在start到end-1位置间的元素，从start开始，以步长step开始取。step默认为1，表示连续获取。

5.拷贝1个list

浅拷贝是在开辟了另一块空间或创建另一个容器，但容器内元素的地址只是源对象复制了一份过来，俗称新瓶装旧酒，深拷贝则是创建另一片地址空间上的容器，且容器内元素的地址与源对象无关，只是值暂时相等，俗称新瓶装新酒，且酒味暂时一致。
