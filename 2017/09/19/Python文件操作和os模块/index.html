文件读写操作

在使用Python的时候常常和文件打交道，为了避免老是打开文件夹手动操作，小小总结了一下常用的一些文件操作。

各个参数的用法：

1.r，只读模式(默认)，不可更改文件
2.w，只写模式，不存在则创建，存在则删除原先内容再写，不可读，即不能read
3.a，追加内容，可读，不存在则创建，存在则追加内容
4.+，表示可以同时读写，w+==w
5.b，处理二进制文件，文本文件不需要。

1.read()
