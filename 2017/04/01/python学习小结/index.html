enumerate用法

a = enumerate([])

import numpy as np

a = np.array(([1,5,10,100],[5,7,1,8],[9,2,4,5])) /接受一个元组参数
array([[ 1, 5, 10, 100],
[ 5, 7, 1, 8],
[ 9, 2, 4, 5]])

a.min()返回矩阵中最小的一个值
a.min(0)返回每列中最小的值，是一个1×n一维数组
a.min(1)返回每行中最小的值，1×n一维数组

max()函数同理

a.shape #返回一个元组(hang,gao)
a.shape[0] 返回行数
a.shape[1] 返回列数

对于mn矩阵a, a.sum(axis=0)为列相加，形成1n
a.sum(axis=1)为行相加，形成1*m

对于1*n矩阵，例如a[7,4,2,3],a.argsort()返回元素从小到大排列的索引值元组
